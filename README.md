## ✨ مميزات التطبيق

| الميزة | التفاصيل |
|--------|----------|
| 🕐 **مواقيت الصلاة** | GPS تلقائي + يدوي + 10 طرق حساب |
| 📢 **الأذان التلقائي** | يعمل في الخلفية تلقائياً عند وقت كل صلاة |
| 🎵 **صوت مختلف لكل صلاة** | إمكانية تخصيص الأذان لكل وقت |
| 🔊 **تحكم في الصوت** | شريط تحكم كامل في الصوت |
| 📿 **الأذكار** | صباح / مساء / بعد الصلاة + تسبيح |

| 🧭 **القبلة** | بوصلة GPS حقيقية + المسافة للكعبة |
| ⚙️ **الإعدادات** | حفظ كامل للإعدادات |
| 📅 **التقويم الهجري** | يظهر التاريخ الهجري مع اليوم |

## 📸 صورة الوالد

لإضافة صورة الوالد رحمه الله:
1. ارفع صورة بصيغة `.jpg` أو `.png`
2. اسمها: `father_photo.png`
3. ضعها في: `app/src/main/res/drawable/`
4. **ستظهر في كل شاشات التطبيق تلقائياً** ✅

---

## 🎵 إضافة صوت الأذان

1. ارفع ملف صوتي MP3
2. اسمه: `athan_default.mp3`
3. ضعه في: `app/src/main/res/raw/`
4. يمكن إضافة أذان مختلف لكل صلاة:
   - `athan_fajr.mp3`
   - `athan_dhuhr.mp3`
   - `athan_asr.mp3`
   - `athan_maghrib.mp3`
   - `athan_isha.mp3`

---

## 🚀 بناء التطبيق

1. ارفع المشروع على GitHub
2. اذهب إلى **Actions** → **Build Islamic App APK**
3. اضغط **Run workflow**
4. بعد اكتمال البناء → **Artifacts** → حمّل الـ APK

## 🤝 إشعارات الخلفية - لا تعارض

| الموقف | التصرف |
|--------|--------|
